## プルトとは？

プルトは**エージェントアプリケーションを作るための開発フレームワーク**です。アプリをパッケージ化するための標準を定義し、どのコードエージェント（Claude Code、Gemini CLI、Codex、Windsurf）でも追加設定なしに理解・実行できます。

核心的なアイデア：**リポジトリがアプリケーション**です。エージェントはリポジトリを読むだけで、自分が誰で、何ができて、どう動くべきかを知ります。

## クイックスタート

:::note
プルトをエージェント経由で使うには、SKILL.md互換エージェント（Claude Code、Gemini CLI、Codex、Windsurf など）が必要です。CLIだけでジェネレーター/バリデーターを使うフローも可能です。
:::

```bash
# 1. フレームワークをクローン
git clone https://github.com/pepetox/puruto.git && cd puruto

# 2. ジェネレーターの依存関係をインストール
pip install jinja2

# 3. エージェントで開いてエコシステムを初期化
/init

# 4. 最初のアプリを作成
/puruto-generator
```
